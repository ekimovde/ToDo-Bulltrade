# Vue 3 + Vite

Особенности приложения:
- TODO-приложение для создания заметок.
- Стек: Vue3, Vite, Vuex, VueRouter
- Добавление задач
- Удаление задач
- Редактирование задачи
- Удаление задачи
- Drag and Drop задач

Инструкция по запуску приложения:
- git clone https://github.com/ekimovde/ToDo-Bulltrade.git
- cd ToDo-Bulltrade
- yarn
- yarn dev
